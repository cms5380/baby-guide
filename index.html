0개월

신생아 (출생~4주)

세상과의 첫 만남, 적응의 시간
⭐ 발달
  • 시야 20~30cm 거리만 흐릿하게 보임
  • 원시 반사: 모로 반사, 빨기 반사, 움켜잡기 반사
  • 소리에 반응하여 몸을 움찔함
  • 엄마 목소리를 구별할 수 있음
🍼 수유
  • 1일차: 약 10ml / 2일차: 20~30ml
  • 3일 이후: 1회 40~60ml, 하루 8~12회
  • 2~3시간마다 수유 (모유 기준)
  • 분유: 2주 이후 80~120ml x 7~8회
🌜 수면
  • 하루 총 16~18시간 수면
  • 2~4시간 간격으로 깨어남
  • 밤과 낮의 구분이 아직 없음
  • 항상 바로 눕혀 재우기 (SIDS 예방)
💉 예방접종
  • BCG (결핵) - 생후 4주 이내
  • B형간염 1차 - 출생 직후
  • 산모가 B형간염 양성 시 12시간 이내 접종 필수
🎁 추천 제품
  • 필수 속싸개/스와들 - 모로반사 방지, 안정감 제공
  • 필수 젖병 + 소독기 - 광구형 160ml 2~3개
  • 필수 카시트 (바구니형) - 퇴원 시부터 필수
  • 추천 흑백 초점책/모빌 - 20~30cm 시야 자극
  • 추천 바운서 - 달래기 & 부모 양손 확보
  • 추천 수유쿠션 - 올바른 수유 자세 유지
1개월

생후 1개월

서서히 세상에 관심을 보이기 시작
⭐ 발달
  • 엎드려 놓으면 잠깐 고개를 들 수 있음
  • 움직이는 물체를 눈으로 따라가기 시작
  • 옹알이의 전 단계인 울음으로 의사소통
  • 엄마 얼굴에 집중하여 바라봄
🍼 수유
  • 모유: 2~3시간마다, 하루 8~12회
  • 분유: 1회 120~140ml x 6~7회
  • 하루 총 수유량 약 840~1000ml
  • 수유 후 트림을 꼭 시켜줄 것
🌜 수면
  • 하루 총 15~16시간 수면
  • 밤잠 8.5시간 + 낮잠 약 7시간 (3~4회)
  • 낮과 밤의 구분이 조금씩 시작됨
  • 수면-수유 패턴을 잡아가는 시기
💉 예방접종
  • B형간염 2차 - 생후 1개월
🎁 추천 제품
  • 필수 유모차 - 바시넷형(눕힘), 생후 1개월부터 외출 가능
  • 필수 아기띠/슬링 - 밀착 육아, 외출 필수
  • 추천 컬러 모빌 - 시야 발달, 흑백→컬러 전환기
  • 추천 플레이짐/아기체육관 - 누워서 손뻗기 놀이
  • 추천 딸랑이 - 청각 자극, 손 쥐기 연습
2개월

생후 2개월

첫 미소와 소통의 시작
⭐ 발달
  • 사회적 미소가 나타남 (의도적 미소)
  • 엎드린 자세에서 고개를 45도까지 들 수 있음
  • 손을 펴고 움켜쥐는 동작 시작
  • "아~", "우~" 등 모음 옹알이 시작
🍼 수유
  • 모유: 하루 6~8회, 한쪽 10~15분씩
  • 분유: 1회 140~160ml x 6회
  • 하루 총 수유량 약 960~1080ml
  • 수유 간격이 조금씩 늘어남
🌜 수면
  • 하루 총 14~16시간 수면
  • 밤에 더 긴 시간 연속 수면 가능
  • 낮잠 3회 정도
💉 예방접종
  • DTaP 1차 (디프테리아/파상풍/백일해)
  • IPV 1차 (폴리오)
  • Hib 1차 (뇌수막염)
  • PCV 1차 (폐렴구균)
  • 로타바이러스 1차 (경구)
🎁 추천 제품
  • 추천 손목 딸랑이/발목 딸랑이 - 손발 인식 촉진
  • 추천 사운드북 - 동물소리, 자장가 등 청각 자극
  • 추천 헝겊 인형 - 촉감 탐색, 애착 형성
3개월

생후 3개월

활발한 소통과 손 탐색
⭐ 발달
  • 엎드린 상태에서 팔로 상체를 지탱
  • 손을 쳐다보며 탐색 (핸드리가드)
  • 소리 나는 방향으로 고개를 돌림
  • 옹알이가 더 다양해짐
  • 웃으며 소리 내어 웃기 시작
🍼 수유
  • 분유: 1회 160~200ml x 5~6회
  • 모유: 하루 약 750~800ml 섭취
  • 수유 간격 3~4시간으로 규칙적
🌜 수면
  • 밤잠 10시간 + 낮잠 5시간 (3회)
  • 총 수면시간 약 15시간
  • 밤에 5~6시간 연속 수면 가능
  • 수면 루틴 만들기 좋은 시기
💡 육아 팁
  • 타미타임(엎드려 놀기)을 하루 여러 번 시도
  • 딸랑이 등 소리나는 장난감으로 자극
  • 아기와 눈을 맞추며 대화하기
🎁 추천 제품
  • 필수 타미타임 매트 - 엎드려 놀기 전용, 거울 포함
  • 추천 오볼(O-ball) - 잡기 쉬운 구조, 국민 장난감
  • 추천 치발기 - 구강 탐색기 시작 대비
  • 추천 촉감책/헝겊책 - 바스락 소리, 다양한 질감
4개월

생후 4개월

목 가누기 완성, 수면 퇴행 주의
⭐ 발달
  • 목 가누기 완성 (고개를 안정적으로 지지)
  • 물건을 잡고 입으로 가져감
  • 양손으로 물건을 잡을 수 있음
  • 감정 표현이 다양해짐 (기쁨, 불만)
🍼 수유
  • 분유: 1회 180~200ml x 5회
  • 하루 총 800~1000ml
  • 분유 아기: 이유식 시작 가능 시기 (4~6개월)
  • 모유 아기: 6개월까지 완전모유수유 권장
🌜 수면
  • 밤잠 11시간 + 낮잠 3.5시간 (2~3회)
  • 총 수면시간 약 14~14.5시간
  • 4개월 수면 퇴행 발생 가능
  • 일관된 수면 루틴이 특히 중요
💉 예방접종
  • DTaP 2차
  • IPV 2차
  • Hib 2차
  • PCV 2차
  • 로타바이러스 2차
🎁 추천 제품
  • 필수 점보 매트/폴더매트 - 뒤집기 시작, 바닥 안전 확보
  • 추천 쏘서/점퍼루 - 다리 힘 기르기, 흥미 유발
  • 추천 소리나는 공 - 잡고 흔들기 놀이
  • 추천 아기 거울 - 자아 인식, 배밀이 자극
5개월

생후 5개월

뒤집기 도전과 호기심 폭발
⭐ 발달
  • 뒤집기 시도 및 성공 (앞에서 뒤로)
  • 발을 잡고 입으로 가져감
  • 한 손에서 다른 손으로 물건 옮기기
  • 거울 속 자신에게 반응
🍼 수유
  • 분유: 1회 200~220ml x 4~5회
  • 이유식 시작 가능 (쌀미음부터)
  • 이유식은 하루 1회, 1~2 숟가락부터
  • 새 식재료는 3~5일 간격으로 추가
🌜 수면
  • 밤잠 11시간 + 낮잠 약 3시간 (2~3회)
  • 밤중 수유 횟수가 줄어듦
  • 뒤집기로 인해 수면 방해 가능
💡 육아 팁
  • 뒤집기 시작 후 침대 안전 재점검
  • 이유식 알레르기 반응 관찰 (발진, 설사 등)
  • 다양한 질감의 장난감 제공
🍚 이유식 재료 (초기 1단계)

미음 형태 / 하루 1회 / 1~2숟가락부터

  • 곡류 쌀 (첫 이유식은 쌀미음으로 시작)
  • 채소 애호박, 감자, 고구마, 브로콜리, 청경채
  • 과일 사과, 배 (익혀서 곱게 갈기)

새 재료는 3~5일 간격으로 1가지씩 추가

🎁 추천 제품
  • 필수 범보의자/부스터 - 앉기 연습 보조
  • 추천 치발기 (냉장 보관용) - 이앓이 시작 대비
  • 추천 컵쌓기 (스태킹컵) - 대소 개념, 무한 활용
  • 추천 물놀이 장난감 - 목욕시간 흥미 유발
6개월

생후 6개월 (반돌)

이유식 본격 시작, 앉기 시도
⭐ 발달
  • 도움 받아 앉기 가능
  • 뒤에서 앞으로 뒤집기도 가능
  • 손으로 물건을 쥐고 흔들 수 있음
  • 자음이 포함된 옹알이 ("바바", "마마")
  • 낯가림이 시작될 수 있음
🍼 수유
  • 분유: 1회 200~220ml x 4회
  • 이유식 하루 1~2회 본격 시작
  • 쌀죽 → 야채죽 → 과일, 단백질 순으로
  • 모유 아기도 6개월부터 이유식 시작
  • 물이나 보리차 소량 시작 가능
🍚 이유식 재료 (초기 2단계)

미음→묽은 죽 전환 / 하루 1~2회

  • 곡류 쌀, 오트밀, 찹쌀
  • 채소 양배추, 비타민, 무, 단호박, 완두콩
  • 과일 바나나, 자두, 아보카도
  • 단백질 소고기 (철분 보충, 곱게 갈아서)

⚠ 꿀, 생우유, 견과류 절대 금지

7~8개월

생후 7~8개월

혼자 앉기와 이앓이의 시작
⭐ 발달
  • 혼자 앉기 가능 (7개월 전후)
  • 배밀이(기기의 전 단계) 시작
  • 엄지와 다른 손가락으로 물건 집기

  • 이가 나기 시작 (앞니 아래 2개)
🍼 수유
  • 이유식 하루 2회 + 수유 3~4회
  • 으깬 음식에서 작은 덩어리 음식으로 전환

  • 손으로 집어먹기(핑거푸드) 시도

잘게 다진 죽 / 하루 2회 / 입자감 도입

  • 해산물 흰살생선(대구, 가자미), 미역, 김

9~10개월

생후 9~10개월

기기와 붙잡고 서기, 탐험의 시작

  • 기기(하이하이) 시작
